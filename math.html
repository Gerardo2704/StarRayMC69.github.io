  1. A function is an expression, rule, or law that defines a relationship between one variable (the independent variable) and another variable (the dependent variable). A ‘function’ must pass the vertical line test to be considered a function.
  2. Functions contain a set of independent domain values and a set of dependent range values.
  3. The domain is the independent variable in a function, it doesn’t change in a result of another variable’s change, in a graph this is typically represented by the x-axis, while the range is the dependent variable which is affected by the domain, it is represented by the y-axis in a graph.
  4. You can determine a function if it’s highest degree is 1, or you can use the vertical line test to check whether all x values correspond to only one y-value, and not two.
  5. The vertical line test checks whether the highest degree of a graphed relation is equal to 1. If you can draw any vertical line that intersects more than one point on the relationship, then it is not a function.
  6. The expression "f (x)" means "a formula, named f, has x as its input variable". It does not mean "multiply f and x".

Calculate:

  • What will the temperature be at 100m depth?
    h = 100m
    t = 15 + 0.01h
    t = 15 + 0.01(100)
    t = 15 + 1
    t = 16

    16 Degrees Celsius
  • Based on the equation, at what depth would there be a temperature of 100C?
    t = 100 C
    t = 15 + 0.01h
    100 = 15 + 0.01h
    100 - 15 = 15 - 15 + 0.01h
    75 = 0.01h
    75 ÷ 0.01 = 0.01h ÷ 0.01
    7500 = h

    7500m
